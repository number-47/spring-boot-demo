# spring boot 与 Mybatis的整合
对User表的Crud
 
Rest方式：

| 请求方式 | url            | 功能说明   |
| -------- | -------------- | -------------- |
| get      | /getUser/id    | 根据id获取用户 |
| post     | /addUser       | 增加用户   |
| put      | /updateUser    | 根据id更新用户 |
| delete   | /deleteUser/id | 根据id删除用户 |
